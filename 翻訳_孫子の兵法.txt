ご提示いただいた資料『訓訳 孫子兵法』の全13篇の内容を、現代語として読みやすい形で全文書き起こします。

### 1. 

計篇（けいへん） 

孫子は言う。戦争は国家の大事であり、生か死か、存続か滅亡かの分かれ道であるから、深く察しなくてはならない 。
これを分析するには五つの事柄（五事）を基本とし、計数によって実情を探る 。一つ目は「道」、二つ目は「天」、三つ目は「地」、四つ目は「将」、五つ目は「法」である 。

* 
**道**：民衆と指導者の意志を一つにすること。そうすれば、民は生死を共にし、危険を恐れなくなる 。


* 
**天**：陰陽、寒暑、時節などの自然現象 。


* 
**地**：距離、険しさ、広さ、地形の利不利 。


* 
**将**：知略、信頼、仁慈、勇気、厳格さ 。


* 
**法**：軍の編成、役職のルール、軍需物資の管理 。



これら五つは、将軍として知らない者はいないが、真に理解している者が勝ち、そうでない者は勝てない 。そこで、以下の項目を比較して実情を探る。主君はどちらが正しい道を行っているか、将軍はどちらが有能か、天地の利はどちらにあるか、法令はどちらが徹底されているか、軍勢はどちらが強いか、兵士はどちらが訓練されているか、賞罰はどちらが明確か 。私はこれによって勝負を知るのである 。

私の計略を聞き入れるなら、これを用いれば必ず勝つので私は留まる。聞き入れないなら、必ず敗れるので私は去る 。計略が利として受け入れられれば、「勢」を形成して外部の助けとする。勢とは、有利な条件に基づいて臨機応変に主導権を握ることである 。

兵法とは「詭道（きどう）」、つまり欺きである 。能力があっても不能に見せ、必要であっても不要に見せ、近くにいても遠くに見せる 。利で誘い、混乱に乗じて奪い、充実していれば備え、強ければ避け、怒らせてかき乱し、謙虚なふりをして相手を慢心させ、休んでいるなら疲れさせ、親密なら引き離す 。備えがない場所を攻め、思いがけない場所に出る。これが兵家の勝ち方であり、あらかじめ固定して伝えることはできない 。

戦う前の「廟算（作戦会議）」で勝つ者は、勝利の計算が多い。勝てない者は計算が少ない。計算が多い方が勝ち、少ない方は勝てず、計算が全くない場合は言うまでもない。私はこれを見て勝負を見抜くのである 。

### 2. 

作戦篇（さくせんへん） 

孫子は言う。軍隊を動かすには莫大な費用がかかる。戦車、武装兵10万、千里の食糧輸送、内外の雑費や接待費、資材の補充などで、一日に千金を費やす。その後にようやく10万の軍は立ち上がるのである 。
戦争が長引けば、兵は疲れ、鋭気はくじかれ、城を攻めても力は尽きる。長く軍を外に出せば国の財政が不足する 。そうなれば諸侯がその弱みに乗じて立ち上がり、知者がいても後始末はできない。ゆえに、戦争には「拙速（不出来でも速いこと）」が良いとは聞くが、「巧久（上手で長引くこと）」が成功した例はない 。戦争を長引かせて国に利益があったことは一度もない 。

戦上手な者は、兵役を二度は課さず、食糧も三度は運ばない。軍需品は国から出すが、食糧は敵から奪う。そうして軍の食糧を足らせるのである 。国が戦争で貧しくなるのは、遠くへ輸送するからだ。遠くに運べば民衆は貧しくなり、軍の近くでは物価が上がり、民の財産は尽きる 。財産が尽きれば重い負担に苦しみ、家庭は空っぽになり、民の財産の七割、公費の六割が失われる 。ゆえに智将は敵の食糧を食べる。敵の一鍾を食べるのは、自国から運ぶ二十鍾に相当する 。

敵を殺すのは怒りによるものであり、敵の財物を取るのは褒美という利によるものだ 。車戦で敵の車を十台以上奪えば、最初に奪った者を賞し、旗を付け替えて味方の車と混ぜて使い、捕虜は手厚く養う。これこそ敵に勝って自分を強くすること（勝敵益強）である 。
ゆえに兵は勝利を尊び、長期戦を尊ばない。兵法を知る将軍は、民の命を司り、国家の安危を握る主宰者である 。

### 3. 

謀攻篇（ぼうこうへん） 

孫子は言う。用兵の法は、敵国を傷つけずそのまま降伏させる（全つ）のが最上で、打ち破るのは次善である 。軍・旅・卒・伍の単位でも同様に「全つ」のが上である。ゆえに百戦百勝が最高なのではない。戦わずして敵を屈服させるのが最高（善の善）なのである 。

ゆえに、最上の戦術は敵の「謀（計略）」を討つこと、その次は「交（同盟）」を討つこと、その次は「兵（軍隊）」を討つこと、最下策は「城」を攻めることである 。攻城はやむを得ない場合のみ行う。準備に三ヶ月、土塁を築くのにさらに三ヶ月かかる 。将軍が怒りに勝てず兵を蟻のように城壁に取り付かせれば、三分の二の兵を失っても城は落ちない。これが攻城の災いだ 。
ゆえに、戦上手な者は、戦わずに敵兵を屈服させ、攻めずに城を抜き、長引かせずに国を滅ぼす。必ず「全」をもって天下に争う。ゆえに兵を疲れさせずに利益を全うできる。これが謀攻の法である 。

将軍は国の補佐役であり、その補佐が完璧なら国は強く、隙があれば国は弱い 。君主が軍に害を与えることが三つある。

1. 進むべきでないのに進めと言い、退くべきでないのに退けと言う。これを「軍を摩る（足かせをはめる）」と言う 。


2. 軍事を知らないのに軍政に干渉すれば、兵士は惑う 。


3. 軍の臨機応変な権限を知らないのに指揮を同じくすれば、兵士は疑う 。
兵が惑い疑えば、諸侯が難を仕掛けてくる。これを「軍を乱して勝利を引き寄せる」と言う 。



勝利を知るには五つの条件がある 。

1. 戦うべき時と戦わざるべき時を知る者は勝つ 。


2. 人数の多寡に応じた戦い方を知る者は勝つ 。


3. 上下の意志が一致している者は勝つ 。


4. 準備を整えて、準備のない敵を待つ者は勝つ 。


5. 将軍が有能で、君主が干渉しない者は勝つ 。
これらが勝利を知る道である。ゆえに言う。「彼を知り己を知れば、百戦して殆（あや）うからず。彼を知らずして己を知れば、一勝一負す。彼を知らず己も知らざれば、戦う毎に必ず殆うし」 。



### 4. 

形篇（けいへん） 

孫子は言う。昔の戦上手は、まず自分が負けない態勢を整えてから、敵が勝たせてくれる隙を待った 。負けないことは自分次第だが、勝てるかどうかは敵次第である。ゆえに、勝つことを知ることはできるが、無理やり勝つ状況を作ることはできない 。
負けない態勢を作るのは「守り」であり、勝機を掴むのは「攻め」である。守る時は九地の下に隠れるように静かに、攻める時は九天の上に動くように激しく。そうして自分を保ち、完全な勝利を得るのである 。

誰にでも見える勝利を収めるのは最高ではない。天下の人が皆「善い」と言う勝ち方は最善ではない 。それは、秋の細い毛を持ち上げるのが力持ちと言われず、太陽や月を見るのが目が良いと言われず、雷鳴を聞くのが耳が良いと言われないのと同じだ 。昔の戦上手は、勝ちやすい敵に勝つ。ゆえにその勝利には、知略の名声も、勇猛な功績も目立たない 。その勝利に間違いがないのは、すでに敗北が決まっている相手に勝つからである 。ゆえに戦上手は、不敗の地に立って、敵が敗れる隙を逃さない。勝利する軍はまず勝ってから戦いを求め、敗北する軍はまず戦ってから勝ちを求める 。

兵法には五つの段階がある。一に「度（土地の広さ）」、二に「量（物資の量）」、三に「数（兵の数）」、四に「称（戦力の比較）」、五に「勝（勝利の確信）」である 。土地から物資が、物資から数が、数から比較が、比較から勝利が生まれる 。
勝利する軍が民を戦わせるのは、せき止めた水を深い谷底へ一気に決壊させるようなものである。これが「形」である 。

### 5. 

勢篇（せいへん） 

孫子は言う。大人数を少人数と同じように治めるのは「分数（組織構成）」による 。大人数を少人数と同じように戦わせるのは「形名（指揮伝達）」による 。全軍が敵を受けても敗れないようにするのは「奇正（正攻法と奇襲）」による 。軍を投入して、石を卵にぶつけるように粉砕するのは「虚実」による 。

戦いは「正」で合流し、「奇」で勝つ 。戦いの勢いは奇正の組み合わせであり、その変化は天地のように窮まりがなく、川の流れのように尽きることがない 。
激流が石を流すのが「勢」であり、猛禽が獲物を打ち砕くのが「節（タイミング）」である 。ゆえに戦上手は、勢いは険しく、節は短く。勢いは弓を引き絞った状態であり、節は引き金を発する瞬間である 。

混乱は規律から、臆病は勇気から、弱さは強さから生まれる 。規律か混乱かは組織の問題（数）であり、勇気か臆病かは勢いの問題（勢）であり、強さか弱さは態勢の問題（形）である 。
上手く敵を動かす者は、形を見せて敵を従わせ、餌を与えて敵を誘い出し、利で動かして待ち伏せで叩く 。
ゆえに戦上手は、勝利を「勢」に求め、個々の部下を責めない。人を適材適所に選び、勢いに任せる 。勢いに任せるとは、木や石を転がすようなものだ。木石は安定していれば静止し、不安定なら動き、四角なら止まり、丸ければ転がる。人を戦わせる勢いを、丸い石を千尋の山から転がすようにさせるのが「勢」である 。

### 6. 

虚実篇（きょじつへん） 

孫子は言う。先に戦場に着いて敵を待つ者は余裕があり（佚）、遅れて戦場に駆けつけ戦う者は疲れる（労） 。ゆえに戦上手は、敵を思い通りに動かし（人を致す）、敵の思い通りに動かされない（人に致されず） 。敵を自ら来させるのは利で誘うからであり、敵が来られないようにするのは害で阻むからだ 。
敵が余裕があれば疲れさせ、満腹なら飢えさせ、落ち着いていれば動かす。敵が必ず駆けつける場所へ出向き、敵が思いもしない場所へ急行する 。

千里を行軍しても疲れないのは、敵のいない場所を行くからだ。攻めて必ず取るのは、守りのない場所を攻めるからだ。守って必ず堅いのは、攻められない場所を守るからだ 。
ゆえに、攻めるのが上手い者は、敵にどこを守ればいいか分からせない。守るのが上手い者は、敵にどこを攻めればいいか分からせない。微細にして形なく、神秘にして声もなし。これによって敵の運命を支配するのである 。

進んで防げないのは、敵の「虚」を突くからだ。退いて追えないのは、速すぎて及ばないからだ 。
私が戦おうと思えば、敵が塁を高くし溝を深くしても、戦わざるを得なくなる。それは敵が必ず救わなければならない場所を攻めるからだ 。私が戦いたくなければ、地面に線を引いて守るだけでも、敵は攻めてこられない。それは敵が行こうとする方向をそらすからだ 。

敵には形を現させて、こちらは形を見せない。そうすれば、こちらは一つに固まり、敵は分散する 。こちらは全力を一つに集中し、敵が十に分かれれば、十で一を攻めることになる。ゆえにこちらは多人数、敵は少人数となり、少人数を撃てば戦いは楽になる 。
私が戦う場所を敵に知られなければ、敵は多くの場所を備えなければならず、備える場所が多ければ、実際に戦う兵は少なくなる 。前を備えれば後が薄くなり、左を備えれば右が薄くなる。至る所を備えれば、至る所が薄くなる 。
「少」なのは敵に備えるからであり、「多」なのは敵に備えさせるからである 。

兵の形は「水」に象る。水は高いところを避けて低いところへ流れる。兵は「実（備え）」を避けて「虚（隙）」を撃つ 。水は地形によって流れが決まり、兵は敵によって勝ちが決まる。ゆえに兵に固定した勢いはなく、固定した形もない。敵の変化に応じて勝ちを取る者を「神」と言う 。

### 7. 

軍争篇（ぐんそうへん） 

孫子は言う。軍の指揮を執り、衆をまとめ、敵と対峙して勝利を争う「軍争」ほど難しいものはない 。その難しさは「迂（まわり道）」を「直（近道）」とし、「患（わざわい）」を「利」に変えることにある 。
あえて遠回りをして利で敵を誘い、敵より後に出発して敵より先に着く。これが「迂直の計」を知る者である 。

軍を挙げて利を争えば間に合わず、重装備を置いて急げば物資を失う 。無理な強行軍で百里先で利を争えば、将軍が捕虜になる。五十里なら半分しか着かず、三十里なら三分の二しか着かない 。物資や兵糧のない軍は滅びる 。

兵は詐（あざむき）によって立ち、利によって動き、分合によって変化する 。

* 
**疾（はや）きこと風の如く** 


* 
**徐（しず）かなること林の如く** 


* 
**侵掠（しんりゃく）すること火の如く** 


* 
**動かざること山の如し** 


* 
**知り難きこと陰（かげ）の如く、動くこと雷震の如し** 迂直の計を先に知る者が勝つ。これが軍争の法である 。



軍政には「言葉が届かないから鳴り物（鼓金）を使い、見えないから旗（旌旗）を使う」とある 。金鼓・旌旗は兵士の耳目を一つにするためのものだ。心が一つになれば、勇者も独りでは進まず、臆病者も独りでは退かない 。これが大人数を動かす法である。
また、敵の「気」を奪い、将軍の「心」を奪う。朝の気は鋭く、昼はだらけ、夕方は帰りたくなる。戦上手は鋭気を避け、だらけ帰ろうとする敵を撃つ。これが気を治める者である 。

### 8. 

九変篇（きゅうへんへん） 

孫子は言う。用兵の法には原則がある。高い丘には向かわず、背後が丘の敵には逆らわず、袋小路（絶地）には留まらず、逃げるふり（佯北）には乗らず、精鋭は攻めず、囮（餌兵）は食べず、帰る敵は遮らず、囲んだ敵には逃げ道を開け、窮地の敵は追い詰めない 。
通ってはならない道、撃ってはならない敵、攻めてはならない城、争ってはならない地、受け入れてはならない君主の命がある 。

知者の考えは、必ず「利」と「害」の両面をあわせて考える。利を考えるから計画が確実になり、害を考えるから災いを取り除ける 。
用兵の法は、敵が来ないことを頼みにするのではなく、いつ来てもいいように自分の備えがあることを頼みにする。敵が攻めてこないことを頼みにするのではなく、自分が攻めさせない態勢であることを頼みにする 。

将軍には五つの危うい性格がある。

1. 
**必死**：死ぬ気で戦うだけの者は殺される 。


2. 
**必生**：生き残ろうとするだけの者は捕虜になる 。


3. 
**忿速**：怒りっぽく短気な者は侮られる 。


4. 
**廉潔**：潔癖すぎて名誉を重んじる者は辱められる 。


5. 
**愛民**：民や兵を慈しみすぎる者は煩わされる 。
これらは将軍の過ちであり、軍を滅ぼす原因となる。深く察しなければならない 。



### 9. 

行軍篇（こうぐんへん） 

軍の進め方と敵の状態の見極めについて。
山を越えるときは谷に沿い、高い場所に陣を置く。川を渡ったら必ず水から遠ざかり、敵が渡ってくるときは半ばまで渡らせて撃つのが利である 。湿地帯（斥沢）は速やかに去る。平地では平坦な場所に処し、高みを右背にする 。

軍は高みを好み低きを嫌い、日向を尊び日陰を忌む。丘陵や堤防は必ず日向を右背にする 。
敵の近くが静かなのは、険阻な地形を頼んでいるからだ。遠くから戦いを挑んでくるのは、進ませようとしているのだ 。

* 多くの木が動くのは、敵が来る徴候 。


* 草むらに仕切りが多いのは、疑わせようとしている 。


* 鳥が飛び立つのは、伏兵がいる 。


* 獣が驚き走るのは、大軍が隠れている 。


* 埃が高く鋭いのは戦車、低く広いのは歩兵が来る徴候 。


* 言葉が低姿勢で備えを増すのは進撃、言葉が強気で進撃する構えは撤退の前触れ 。


* 旗が動くのは乱れている証拠、役人が怒るのは疲れている証拠 。



兵の数は多ければいいというものではない。武力で突き進まず、戦力を集中して敵の状態を判断すれば（敵を料れば）十分である。考えなしに敵を侮る者は必ず捕虜になる 。
兵が親しむ前に罰すれば服従せず、親しんだ後に罰が行われなければ使えない。ゆえに教育（文）でまとめ、規律（武）で斉える。これが必勝の軍である 。

### 10. 

地形篇（ちけいへん） 

地形には「通・挂・支・隘・険・遠」の六つがある 。これらを察するのは将軍の重大な責任である 。
また、将軍の過ちによる敗北が六つある。

1. 
**走**：勢力が互角なのに十倍の敵を撃てば逃げ出す 。


2. 
**弛（し）**：兵が強く役人が弱いと、規律が緩む 。


3. 
**陥**：役人が強く兵が弱いと、軍は崩壊する 。


4. 
**崩**：部下が将軍に不満を持ち、勝手に戦えば崩れる 。


5. 
**乱**：将軍が弱く厳格でなく、教えが不明確なら乱れる 。


6. 
**北（ほく）**：敵を正しく判断できず、弱兵で強敵を撃てば敗走する 。



地形を熟知し、勝算があるなら主君が戦うなと言っても戦ってよい。勝算がないなら主君が戦えと言っても戦わなくてよい 。名誉を求めず、罪を避けず、ただ民と国の利益を考える将軍は国の宝である 。
兵士を赤ん坊や愛児のように見れば、彼らは深い谷へも行き、共に死んでくれる。しかし、甘やかすだけで命令を聞かせられないなら、使いものにならない 。
「彼を知り己を知れば勝利に危うさはなく、地を知り天を知れば勝利を全うできる」 。

### 11. 

九地篇（きゅうちへん） 

戦う場所の性質による九つの分類。

1. 
**散地**：自国領内で戦う場所。戦ってはいけない 。


2. 
**軽地**：敵地に入ってまだ浅い場所。留まってはいけない 。


3. 
**争地**：奪い合う利のある場所。攻めてはいけない 。


4. 
**交地**：交通の要所。連絡を絶ってはいけない 。


5. 
**衢地（くち）**：諸国と隣接する場所。同盟を結ぶ 。


6. 
**重地**：敵地に深く入り込んだ場所。略奪して食糧を確保する 。


7. 
**圮地（ひち）**：山林や湿地などの進みにくい場所。通り過ぎる 。


8. 
**囲地**：入り口が狭く出口が遠い場所。計略を用いる 。


9. 
**死地**：必死に戦わなければ滅びる場所。ただ戦うのみ 。



戦上手は、敵の連携を断ち、上下を救いあえなくさせ、兵を分散させる 。
兵の心理は速さを尊ぶ。相手が油断している隙に、思いがけない道から、備えのない場所を攻める 。兵を逃げ場のない場所に投げ込めば、死んでも逃げ出さず、士人は全力を尽くす 。
将軍の仕事は、静かに深く、正しく厳格であること。兵の耳目を欺き、計画を知られないようにし、居場所やルートを変えて予測させない 。兵を死地に追い込んでこそ、その後に生路が開けるのである 。

### 12. 

用間篇（ようかんへん） 

孫子は言う。十万の軍を動かせば日に千金を費やす。それなのに、爵位や金を惜しんで敵の情勢を知ろうとしないのは、不仁（思いやりがない）の至りである 。
勝利をもたらす「先知（予知）」は、神仏に祈るものではなく、過去の事例から推測するものでもない。必ず人（間諜・スパイ）を通じて得るものである 。

間（スパイ）には五つある 。

* 
**郷間**：敵の郷里の人を利用する 。


* 
**内間**：敵の官人を利用する 。


* 
**反間**：敵のスパイを寝返らせて利用する 。


* 
**死間**：偽情報を敵に流し、発覚して殺されることを前提とする 。


* 
**生間**：生きて帰って報告する 。



五つのスパイを同時に動かし、その手口が漏れないことを「神紀」と言う 。三軍の中でスパイほど親しくすべき者はなく、報奨は最も厚くし、秘密は最も守るべきだ 。
明主や賢将だけが、高度な知恵を持つ者をスパイとして使い、大きな功績を上げる。これが兵の要である 。

### 13. 

火攻篇（かこへん） 

火攻めには五つの対象がある。人、積み藁、輜重（荷物）、倉庫、隊列である 。火を放つには乾燥した天候や特定の風が吹く日を選ぶ 。
火攻めは攻撃を助ける「明」であり、水攻めは「強」である。水は敵を分断できるが、物資を奪うことはできない 。

最後に重要な戒めがある。
戦って勝ち、攻めて取っても、その目的を達しなければ「費留（無駄）」であり、凶である 。明主はこれを深く考え、良将はこれを実行する。利がなければ動かず、得られるものがなければ用いず、危うくなければ戦わない 。
主君は怒りに任せて軍を起こしてはならず、将軍は不満に任せて戦ってはならない 。利に合えば動き、合わなければ止める。怒りは喜びに、不満は楽しみに戻ることもあるが、滅びた国は二度と存続せず、死んだ者は二度と生き返らないからだ 。
ゆえに明主は慎重になり、良将は警戒する。これこそが国を安んじ軍を全うする道である 。
